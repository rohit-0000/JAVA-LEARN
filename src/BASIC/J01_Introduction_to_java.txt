JVM,JRE and JDK
JVM - Java virtual Machine - specification(virtual machine)
JRE - Java Runtime Environment - it is the implementation of JVM
JDK - Java Development Kit - contains Development tools,JRE

JRE contains JVM and class Libraries
JDK contains JRE + Compilers + Debuggers + JavaDoc
